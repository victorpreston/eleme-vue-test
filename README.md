# Test Coding Elemes.id

This website is a front-end developer coding test at Elemes group

![alt](./cover.png)

## Installation


```bash
# Clone Repo
git clone https://github.com/victorpreston/elemes-vue-test.git
```


```bash
# npm
npm install

# pnpm
pnpm install

# bun
bun install
```

## Running Project


```bash
# npm
npm run dev

# pnpm
pnpm dev

# bun
bun run dev
```
